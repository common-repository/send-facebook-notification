== Description ==
A plugin To send Notification from your WordPress site to Facebook directly with a link for registered user only. To install this plugin SSL certificate (https://)  is needed. I have used Facebook canvas API for this plugin. You are not able to register your site without SSL certificate for Facebook canvas API. 
 Please watch this video to know how to set up Facebook canvas app. 
https://www.youtube.com/watch?v=oqX2oW6NRPE


== Installation ==
This is a very small plugin. So installation process is very easy.SSL certificate is required to install this plugin

1. Install & activate the Plugin. To get an idea about the installation of WordPress plugin you may visit this link.
[Installation help](http://www.wpbeginner.com/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/)

2. Go to  FB Notify menu from Dashboard 

3. Create a Facebook app. Please visit [this link](https://www.youtube.com/watch?v=oqX2oW6NRPE) to know how to create a Facebook app for this plugin.   

4.  Copy & Paste your Credential. Click save.

5. [txp-notification btn-txt=\"Your given text\"]  Copy this shortcode and paste.  You will be able to see a button on the frontend.  your visitor will be able to be listed  with this button for notification


== Frequently Asked Questions ==
Can anyone install this plugin? 

Yes, If your website has SSL certificate, you can install & use this plugin. 

Will I able to see listed people? 

Yes, from plugin settings page you will be able to see the listed people. 

Is this Notification Work for mobile also? 
No, Facebook notification app don\'t support mobile. 

Am I able to download the listed people data? 

Yes, From WordPress backend you will be able to download all the listed people data as CSV.
